### 4.3. Game Mechanics

_Flowchart to be constructed at a later time_

#### Locomotion

* Movement
	* The player will be able to move up, down, left, and right
	* The player should only be able to move within the player area and collide with walls, preventing further movement
* Idle
	* The player state should be idle when not performing any actions (such as: moving or interacting)

#### Interaction

* Talking
	* By interaction with a non-hostile NPC, the player will be able to initiate a conversation with said NPC, bringing up the Dialogue Menu
* Object Collecting
	* Interacting with certain objects in the game will add the object to the inventory, removing it from the map
* Object Manipulation
	* Interacting with certain objects in the game will allow the environment to change
		* Examples:
			* Using a lever to open a hidden path
			* Using a key in the inventory to open a door

#### Dialogue Menu System

* Upon talking to an NPC, a dialogue menu will appear, allowing the player to choose what topics to discuss with said NPC
* Some dialogue options within the dialogue menu will appear/disappear depending if the player has the approperiate item/clue in their inventory
* Dialogue Menu will serve to collect clues from the NPC's and drive the narrative
* Concept Image

	![Dialogue](https://i.ytimg.com/vi/Uc4s79NfovY/maxresdefault.jpg)

#### Inventory System

* GUI
	* The player will be able to see a visualization of objects and clues in an onscreen GUI element
* Description
	* Each object/clue will have a description with how the particular object/clue can be used
* Navigation
	* The inventory system can be navigated
	* The object/clue in the inventory will be highlighted to signify which object/clue is being viewed
* Concept Image

	![Inventory System](https://miro.medium.com/max/225/0*7EVPSkASsXBMDMxQ)

#### Investigation Mode

* Object highlighting
	* Pressing the Investigation key/button will highlight objects that can be interacted with during gameplay
	* Hightlighted objects will have a simple outline around them

#### Spirit/Human Mode

* Plane of Existence Swapping
	* Pressing the Swapping key/button will switch the playable player from human to ghost
		* Swapping between characters will be crucial with investigating and interacting with NPC's and objects to be able to solve puzzles.

### 8.2. Platforms & Monetization

This game will be released, without ads, on the specified platforms (Android and PC), at no cost for the user. This game will not feature micro-transactions or any form of monetization.
